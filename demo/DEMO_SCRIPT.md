## BACKUP PLANS

**If simulation doesn't run:** Use the API directly:
```bash
curl -X POST http://localhost:8000/simulate \
  -H "X-API-Key: fw-demo-key-2026" \
  -d '{"domain":"hiring"}'
```

**If WhatIf shows no bias:** Use hiring domain (not lending/admissions)
and make sure you clicked a REJECTED/female row.

**If score stays at 0:** Backend is not connected.
Open Settings → check Backend URL = `http://localhost:8000` or Cloud Run URL.

**If Gemma explanation is generic text:** That's the template fallback —
set `GEMMA_BACKEND=ollama` and run `ollama serve` for real explanations.
